📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de estudar e praticar automações em Python, utilizando bibliotecas como time e pyautogui.

🎯 Objetivo

Automatizar tarefas repetitivas no computador, como:

Abrir aplicativos automaticamente
Navegar até sites específicos
Enviar mensagens de forma automatizada
Simular interações do usuário (cliques, digitação, etc.)

O projeto serve como base para explorar diferentes possibilidades de automação e aumentar a produtividade em tarefas do dia a dia.
